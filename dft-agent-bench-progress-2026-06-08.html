DFT 简介 · primer
Slide 2 — DFT 是什么 · 为什么需要 Agent

DFT 是材料 / 化学计算的「主语言」，但门槛高、工序长 —— LLM agent 把专家工序变成可自动化的 pipeline。

DFT · Density Functional Theory

从第一性原理出发求解电子结构，算出材料的晶格 / 能量 / 力 / 能带 / 弹性 / 状态方程等物理量。是材料发现、催化、半导体设计的核心计算工具，QE / VASP 是主流引擎。

痛点 · 专家工序长，难规模化

跑一次 QE 要懂 pseudopotential / ecutwfc / k-mesh / SCF 收敛 / smearing 一整套 know-how；建模、调参、跑、解析、判断对错环环相扣，高度依赖专家经验，难以高通量规模化。

新解法 · LLM Agent 端到端

让 agent 读题目 → 写 QE 输入 → 跑 pw.x → 解析输出 → 提交结果，把一长串专家工序变成自动化工作流。问题随之而来：它到底能做到什么档位？需要一把能量化的尺子。

概念：Agent 把题面变成物理量
NL → QE → physics
缺口：现有 LLM bench（SciCode / GPQA / MMLU-Pro）测的是知识 / 代码片段；DFT-Agent Bench 测的是 真起 QE、真出物理量、真打分 —— 第一个把完整 DFT 工作流端到端纳入 LLM 评测的系统化 bench。
听众无需 DFT 背景：核心只有一句 —— agent 读自然语言任务，自己把它变成跑得通、可验证的第一性原理计算

BENCHSlide 5 — 材料维度 M1 / M2 / M3

材料分级 —— 从主族稳定相到磁性强关联，考的是 agent 对体系敏感度的判断：从「默认配置可解」到「必须做物理判断」。

M1 · 简单稳定simple stable
212 题
代表：Si / MgO / Al / Cu / GaAs / Diamond C。考察点：标准 PBE + 基础 ecutwfc / k-mesh / smearing 默认值能否选对。58 材料 × T1+T2 + 52/44 材料 × T3/T4。
M2 · 复杂多元complex multinary
62 题
代表：SiO₂ α-quartz / MgAl₂O₄ spinel / CaTiO₃ / TiO₂ rutile。考察点：多元素 pseudopotential 选型 + 复杂晶胞收敛策略。28 材料 × T1+T2 + 4/2 × T3/T4。
M3 · 磁性 / 强关联magnetic
30 题
代表：Fe bcc / Ni fcc / Cr bcc / Co hcp / NiO / MnO。考察点：必须判断 FM/AFM 初态 + nspin=2 + 自旋极化设置。15 材料 × T1+T2 全档，GT 都跑了 nspin=2 双态对照。
M4 / M5 · 钙钛矿 / 拓扑planned
规划中
化学势 + 缺陷形成能 + 钙钛矿带隙 + 拓扑材料 —— 下一阶段扩展目标。
从 M1 到 M3：不是「加难度」，是能力跃迁
M3 的关键：磁性材料若不开 nspin=2 并设对初始磁矩，会收敛到错误的非磁基态 —— 恰好打中 LLM 当前能力边界。
Web 落地佐证：catalogue 页面按 tier = M1/M2/M3 filter 可直接浏览各档真实题目列表（见 Web 章节）。
材料维度与任务维度解耦 —— 可沿任一轴切片报告 agent 能力，定位强项与短板

① 评测体系 Bench
BENCHSlide 6 — 任务维度 T1 / T2 / T3 / T4

任务分级 —— 从单点静态结构到 24-strain 张量拟合，步数 1 → 24 阶梯式拉满 agent 的工作流闭环、调度与纠错能力。

T1
vc-relax
变胞结构弛豫找平衡态。Agent 须：单次 QE 收敛 + 提取 final lattice。→ 晶格常数 a/b/c + 空间群 number
1 步
T2
bands+DOS
电子结构 —— 材料「指纹」。四步流水线 scf → nscf-bands → bands.x → dos.x，须懂 step 依赖、kpath / nbnd 设对。→ band 数据 + DOS curve + Fermi
4 步链
T3
EOS · BM3
状态方程。7 个 volume 点 × fixed-cell relax，批量调度 7 次 QE + 出 (V,E) 表。→ V0 / B0 / B0'（harness 自己 BM3 拟合）
7 次 QE
T4
elastic Cij
弹性张量。6 voigt mode × 4 amplitude = 24 次 QE 串行 + stress 解析 + 对称化 + Born 稳定性判定。→ 6×6 Cij + K_VRH / G_VRH
24 次 QE
卖点：T1 → T4 步数从 1 → 24，对 agent 的「循环能力 / 错误恢复 / 调度策略」是阶梯式压力测试 —— 长 pipeline 正是模型能力的分水岭。
防 overclaim 设计：T3 不让 agent 自己做 BM3 拟合、T4 不让 agent 自己算 VRH —— agent 只交原始 (V,E) / stress-strain 数据，harness 侧统一拟合打分，保证信号干净。
GT 自跑 wall time 参考：T1 ~5 min · T2 ~30 min · T3 ~1 h · T4 ~5 h（每材料，QE v7.5 GPU backend）

Evaluator workflow · 提交后怎么打分
① Agent submit

↓
② Harness re-fit

↓
③ 多源 GT 比对
re-fit 结果与四源 GT 交叉比对，按字段算相对差。
↓
④ Score ∈ [0,1]
逐字段加权 → total_score；相对差 ≤ 1% 判 pass。每字段可追溯，非黑箱。

1能确定就别赌概率

M2_T2_SiO2_α-quartzM2T2

M3_T1_Fe_bccM3T1

难点：必须开 nspin=2 并判 FM 初态 + 设初始磁矩，否则收敛到错误的非磁基态
