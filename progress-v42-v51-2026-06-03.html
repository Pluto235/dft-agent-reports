1 · 我们在做什么（30 秒背景）

我们在做一个用大模型自动跑 DFT 计算的 agent。通俗讲：DFT（密度泛函理论）是材料计算的主力方法，传统上要专家手写一堆 Quantum ESPRESSO（QE）输入、跑、看结果、反复调。我们让大模型把这套流程自动跑通——给它一道题（某个材料、要算什么），它自己写输入、跑 QE、读输出、检查、交卷。

有一个独立的 bench（测评系统）给每道题自动打分（0–1 分），对照标准答案算每个物理量对不对。所以"agent 强不强"是一个可量化、可对比的分数。下面所有数字都来自这个 bench。

2 · v4.2：通用成熟基线，标准集 27 题 0.918

v4.2 是两条轨道共同的起点，也是目前最成熟的通用版本。它把一道题拆成"规划 → 执行 → 验收"三段流水线，每段由一个大模型角色负责；验收不过就自动重试 / 重新规划。

题目+ 两道开场功课
→
Planner把题拆成步骤
→
Executor写输入 · 跑 QE · 读输出
→
Verifier逐步验收
→
结果交卷打分

验收失败 → 自动重试 / 重新规划。两道"开场功课"（判金属还是绝缘体、先把结构预弛豫一下）由纯代码完成，不劳烦模型。执行环节由模型操作 shell 命令，外加一层"输入自动纠错"（auto-lint）兜底。

核心思想

把 DFT 知识写成"模型绕不过的代码硬约束"

老版本是把 DFT 经验写在提示词里，指望模型照做；v4.2 的关键升级是把这些经验变成代码层的硬规则——交卷前自动检查、自动纠正输入参数、独立验收清单。模型"想偷懒也偷不了"。这让成绩从"看模型心情"变成"稳定可控"。

v4.3 · 手写线的磁性特化版

v4.3 是把 v4.2 直接 fork 出来、专门为磁性 / 金属材料特化的版本。架构和 v4.2 完全一样（同样的"规划→执行→验收"三段式，见上图），区别在于针对磁性材料这类"难收敛、易判错"的体系，加了一组特化处理：

  • 磁序探针：自动试"铁磁 vs 反铁磁"两种磁排列，跑两个单点能量、选更低的那个——磁性材料算对的第一步关键。
  • 金属能带保证：跑完后检查金属费米面附近有没有假"缝"，有就自动加密 k 点重算，避免金属被误判成半导体（典型如 Ni）。
  • 一组磁性工程修复：强制自旋极化（nspin=2）、反铁磁原子标记归一化、把总态密度的两个自旋通道加起来、单条命令墙钟封顶防超时。
  • 顺手修了 v4.2 的边角：例如上一节 Cd 失分（0.445）——v4.3 把它的截断从 50 提到 80 Ry 后回到 0.948。

一句话：v4.3 = 把成熟的 v4.2 针对"磁性材料"这块硬骨头手动打磨过的生产版本；相对 v4.2 多了约 5500 行磁性 / 金属 / 墙钟相关的修复代码。

v5.1 · 全新 SDK 结构化工具底盘

另一条路不再让模型自由拼 shell 命令，而是换成官方 agent SDK 的结构化工具：模型按一组带类型的"按钮"操作（写 QE 输入 / 跑 pw.x / 读输出 …），更安全、更可控、更容易加新能力、更好维护。v5.1 = 这套新底盘 + 把 v4.2 的"材料知识层"搬回来当开场功课。

执行循环旁挂着两个"安全件"：卡死自动断路器、SCF 不收敛时交给一个专门的修复子 agent。这次还把 DeepSeek 也切到了这条 SDK 路径，并修了 4 个评审 bug（把 v5 里两处"物理验收被架空"的地方接回了线）。单元测试 167 → 336 全过。

怎么理解两条线：v4.2 / v4.3 是"手写循环"架构——成熟、单题质量高，但靠不断叠加修复代码维护；v5.1 是"结构化工具"架构——工程上更安全、更可维护、更易扩展，这轮在最新广集上鲁棒性已经反超（12 道零失败）。现阶段两条线各有所长、不急于判定谁取代谁：v4.3 是当前磁性生产主力，v5.1 在 de-risk。

5 · 需要和 DFT 专家杨博确认的问题（4 条）

下面 4 条是我们已经定位到的卡点，但属于"打分口径 / 物理判据"层面，需要杨博拍板。它们也正好解释了为什么 T2 有天花板、而且对两个版本都一样——不是 agent 算错。

问题 1"算出来几乎是零带隙"的金属判据，前后差 100 倍

同一个"带隙接近零"的物理事实，bench 用了两套差 100× 的标准：带隙数值容忍 ±0.1 eV（100 meV），但"是不是金属"的分类判定要 < 1 meV 才算金属。结果 Ge（算出 23 meV）、Rb（20–43 meV）这种近零带隙材料，带隙值判对、但金属/方向/k 点三个分类字段连环判错。

▸ 为什么重要：这类失分不是我们算错，是判据自相矛盾。需要杨博定一个合理的"多小算金属"口径。

问题 2绝缘体的 VBM / CBM 位置拿不到分 → T2 被封顶

所有真有带隙的绝缘体，能带类型 / 价带顶位置 / 导带底位置三个字段现在一律拿不到分（占每题 0.35 权重），等于绝缘体 T2 被封顶在 ~0.65。两个版本都一样。

▸ 为什么重要：这是 T2 分数最大的系统性天花板。需要 bench 侧加一个"把能带极值映射回高对称 k 点标签"的能力，并和杨博确认 k 点匹配口径是不是过严。

问题 3态密度（DOS）的归一化基准

标准答案的 DOS 是"每化学式"归一，QE 默认输出是"每原胞"，对 BN 这种原胞 = 2 个化学式的材料，我们的曲线天然高 2 倍。

▸ 为什么重要：是个一行就能修的口径问题，但要先和杨博确认归一化基准从哪取，避免冤枉失分。

问题 4DOS 费米能 / 曲线的参考系与打分方式

绝缘体的"费米能"字段实际比的是价带顶绝对能量，BN 出现稳定 0.36 eV 的系统偏移（不是抖动），更像"能量零点参考系"没对齐；而 DOS 曲线用面积重叠打分，对"形状对、协议错"太宽容。

▸ 为什么重要：需要杨博定费米能/价带顶的参考系，以及 DOS 曲线该不该换更严的形状度量。
