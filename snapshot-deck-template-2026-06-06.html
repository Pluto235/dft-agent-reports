## 我们在做什么

让大模型 自动跑 DFT 计算：给一道题（某材料 + 要算什么），agent 自己写 Quantum ESPRESSO 输入 → 跑 pw.x → 读输出 → 自检 → 交卷。有一个独立 bench 自动打分 0–1，对照标准答案算每个物理量对不对。所以"强不强"是可量化、可对比的分数。
任务类型
  • T1 = vc_relax · 结构弛豫
  • T2 = bands_dos · 能带 + 态密度
里程碑(难度阶梯)
  • M1 元素 / 简单二元（~60/型）
  • M2 复杂氧化物（28/型）
  • M3 磁性材料（15/型，自旋极化）
这份 deck 的边界
快照 = 三版在 同一张 ~43 题共同卷上的成绩。当前 实验未跑全，主表是 scaffold：已有数据已填、待补格留空——最后一页专门列"下一步补哪些实验"。

## 口径与已知坑

混 build（本快照的主要限制）
v5.1 标准核来自 ac19(落后7提交)、部分 M3 来自 6efa(磁性修复前)——表头写"@ee050a2"但若干格是旧 build。每格已标 build；最干净做法 = P1 在冻结点重跑 v5.1。
v5.1 的 0.683 ≠ 冻结点
那是 6efa15e 的成绩；ee050a2 多了磁性恢复 fbd53da，全集未完整测过。别把 0.683 当 ee050a2 成绩引用。
bench 打分口径 4 条 · 待杨博（非 solver 锅）
  • k-label 非命名点：VBM/CBM 落在非高对称点 → 丢满分（MnO/NiO 实测）
  • 绝缘体 dos_e_fermi 参考系：绝对能量 ±0.1eV 比 → 系统偏移直接判 0（疑致 Fe2O3/Cr2O3-T2 三版皆低）
  • 磁矩绝对 0.5µB 容差：对大磁矩偏松、小磁矩极松，占 T1 权重 0.35
  • null GT：Li₃PO₄ k-label 全 null，null-vs-null 不给分 → 0.2 权重拿不到
→ 报告给所有者 + 杨博确认，不自行改 bench。
另：Ni_fcc 高方差(自旋通道抽签) → 金属格 best-of-3 缓解。跨 user run 目录有 1 个不可读(权限)，收数已跳过。
