为什么选 lossy tier？

Tier每题大小148 题总大小能干啥不能干啥
Minimal~200 KB~30 MBLeaderboard、transcript 详情、看分数看不到能带 / DOS 曲线图
Lossy（本协议）2-4 MB~500 MB上面 + 能带 / DOS 曲线、收敛过程不能重做后处理（没 .wfc）
Fullmedian 1 MB / max 7.7 GB~36 GB完全 bit-exact 复现太大，GH Release 单文件 2 GB 上限需切片

Lossy 在体积和功能上是 sweet spot——单文件能进 GH Release 附件、500 MB 单线程下载 ~1 分钟， 而且 web 该展示的都能展示。如果以后你真的要重做某题的后处理（比如换 dos.x 参数重画 DOS），那时再单独要 full 数据。

§5 export.py 全文

纯标准库 · Python 3.10+ · ~210 行 · 复制保存为 export.py

#!/usr/bin/env python3
"""bench run data exporter — 把 state.db 里指定的 runs + 它们的 run_dir 内容打成 tar.gz。

用法：
    python export.py \\
        --runs-local /home/weilin/bench/runs_local \\
        --filter "agent_id='v1-stable' AND status='done'" \\
        --tier lossy \\
        --out /tmp/bench-bundle-2026-05-13.tar.gz

筛选三种方式（互斥）：
  --filter "<SQL WHERE clause>"   # 任意 SQL 表达式
  --run-ids /path/to/ids.txt        # 每行一个 run_id
  --since 2026-05-08                 # 等价 created_at >= ...

Tier 决定 workdir 里哪些文件打包：
  minimal   只要 report.json + transcript.json + bash.log + result.json + task_prompt.txt
  lossy     上面 + 所有 *.in / *.out / *.dat / lint.log（默认）
  full      所有非 symlink、非 __pycache__ 内容（含 wfc / save，几十 GB）
"""
from __future__ import annotations
import argparse, getpass, gzip, hashlib, json, os, shutil, sqlite3, socket, sys, tarfile, tempfile
from datetime import datetime, timezone
from pathlib import Path

SCHEMA_VERSION = 1
TIERS = ("minimal", "lossy", "full")

# tier → 在 workdir/ 里要保留的文件 glob 模式（其他全部丢）
TIER_WORKDIR_KEEP = {
    "minimal": ["report.json", "task_prompt.txt", "transcript.json",
                "bash.log", "result.json", "lint.log"],
    "lossy":   ["report.json", "task_prompt.txt", "transcript.json",
                "bash.log", "result.json", "lint.log",
                "*.in", "*.out", "*.dat", "*.dat.gnu", "*.dat.rap"],
    "full":    ["*"],
}
# 不论 tier 都要排除的（symlink、大二进制目录、scratch）
ALWAYS_SKIP_NAMES = {"pseudos", "__pycache__", ".save"}
ALWAYS_SKIP_SUFFIXES = (".pyc", ".wfc", ".wfc1")

# run_dir 顶层（workdir 之外）固定保留这些
TOP_LEVEL_KEEP = ["report.json", "harness.log", "harness_stderr.log",
                  "solver_stdout.log", "solver_stderr.log"]


def parse_args():
    p = argparse.ArgumentParser(description="Export bench runs to a transferable tar.gz bundle.")
    p.add_argument("--runs-local", type=Path, required=True,
                   help="path to runs_local/ (contains state.db + run dirs)")
    p.add_argument("--out", type=Path, required=True,
                   help="output bundle path (.tar.gz)")
    p.add_argument("--tier", choices=TIERS, default="lossy",
                   help="bundle tier — controls what's kept in workdir/ (default: lossy)")
    g = p.add_mutually_exclusive_group(required=True)
    g.add_argument("--filter", type=str, metavar="SQL_WHERE",
                   help="SQL WHERE clause (e.g. agent_id='v1-stable' AND status='done')")
    g.add_argument("--run-ids", type=Path, metavar="FILE",
                   help="text file, one run_id per line")
    g.add_argument("--since", type=str, metavar="ISO_DATE",
                   help="created_at >= this date (e.g. 2026-05-08)")
    p.add_argument("--label", type=str, default="",
                   help="free-form label written to manifest.json")
    return p.parse_args()


def select_runs(db_path: Path, args) -> tuple[list[sqlite3.Row], list[str]]:
    con = sqlite3.connect(str(db_path))
    con.row_factory = sqlite3.Row
    # 用 PRAGMA 拿到实际列名（schema 会随 ALTER 漂移）
    cols = [r["name"] for r in con.execute("PRAGMA table_info(runs)")]

    if args.filter:
        sql = f"SELECT * FROM runs WHERE {args.filter}"
        rows = list(con.execute(sql))
    elif args.run_ids:
        ids = [ln.strip() for ln in args.run_ids.read_text().splitlines() if ln.strip()]
        if not ids:
            sys.exit("--run-ids file is empty")
        placeholders = ",".join("?" * len(ids))
        rows = list(con.execute(
            f"SELECT * FROM runs WHERE run_id IN ({placeholders})", ids))
    else:  # --since
        rows = list(con.execute(
            "SELECT * FROM runs WHERE created_at >= ?", (args.since,)))
    con.close()
    return rows, cols


def should_keep(rel_path: Path, tier: str) -> bool:
    """Decide whether to copy this file from a run_dir into the bundle."""
    name = rel_path.name
    if name in ALWAYS_SKIP_NAMES:
        return False
    if any(name.endswith(suf) for suf in ALWAYS_SKIP_SUFFIXES):
        return False
    # rel_path 形如 "workdir/scf.out" 或 "report.json"
    parts = rel_path.parts
    if len(parts) == 1:
        return name in TOP_LEVEL_KEEP
    if parts[0] == "workdir":
        patterns = TIER_WORKDIR_KEEP[tier]
        from fnmatch import fnmatchcase
        return any(fnmatchcase(name, pat) for pat in patterns)
    # workdir/<subdir>/... — lossy/minimal 不递归子目录（除 full）
    return tier == "full"


def stage_run(src_run_dir: Path, dst_run_dir: Path, tier: str) -> tuple[int, int]:
    """Copy filtered files from src to dst. Returns (n_files, total_bytes)."""
    n_files, total_bytes = 0, 0
    for src_path in src_run_dir.rglob("*"):
        if src_path.is_symlink():
            continue   # skip symlinks (pseudos etc.)
        if not src_path.is_file():
            continue
        # skip dirs whose name is in ALWAYS_SKIP_NAMES (e.g. workdir/<prefix>.save/...)
        if any(p in ALWAYS_SKIP_NAMES for p in src_path.parts):
            continue
        rel = src_path.relative_to(src_run_dir)
        if not should_keep(rel, tier):
            continue
        dst_path = dst_run_dir / rel
        dst_path.parent.mkdir(parents=True, exist_ok=True)
        shutil.copy2(src_path, dst_path)
        n_files += 1
        total_bytes += src_path.stat().st_size
    return n_files, total_bytes


def detect_pseudo_dir(runs_local: Path) -> str:
    """猜源 pseudo 目录：第一个 run 的 workdir/pseudos 软链的目标。"""
    for d in sorted(runs_local.iterdir()):
        link = d / "workdir" / "pseudos"
        if link.is_symlink():
            return os.readlink(link)
    return ""


def main():
    args = parse_args()
    db_path = args.runs_local / "state.db"
    if not db_path.exists():
        sys.exit(f"[export] state.db not found: {db_path}")

    print(f"[export] scanning {db_path}")
    rows, cols = select_runs(db_path, args)
    print(f"[export] filter matched {len(rows)} runs")
    if not rows:
        sys.exit("[export] nothing to export — check your filter")

    staging = Path(tempfile.mkdtemp(prefix="bench-bundle-"))
    bundle_root = staging / args.out.stem.replace(".tar", "")   # 去掉双重 .tar.gz 尾缀
    bundle_root.mkdir()
    runs_staging = bundle_root / "runs"
    runs_staging.mkdir()

    db_rows_path = bundle_root / "db_rows.jsonl"
    runs_summary = []
    with open(db_rows_path, "w", encoding="utf-8") as f_jsonl:
        for i, r in enumerate(rows, 1):
            run_id = r["run_id"]
            src = args.runs_local / run_id
            if not src.is_dir():
                print(f"[export] {i:3d}/{len(rows)}  {run_id:60s} → SKIP (run_dir missing on disk)")
                continue
            dst = runs_staging / run_id
            n_files, total_bytes = stage_run(src, dst, args.tier)
            print(f"[export] {i:3d}/{len(rows)}  {run_id:60s} → {n_files:3d} files, "
                  f"{total_bytes/1024/1024:6.2f} MB")
            row_d = {k: r[k] for k in cols}
            f_jsonl.write(json.dumps(row_d, default=str) + "\n")
            runs_summary.append({"run_id": run_id, "task_id": r["task_id"],
                                  "model": r["model"], "status": r["status"],
                                  "total_score": r["total_score"]})

    manifest = {
        "schema_version": SCHEMA_VERSION,
        "tier": args.tier,
        "exported_at": datetime.now(timezone.utc).isoformat(timespec="seconds"),
        "label": args.label,
        "source": {
            "user": getpass.getuser(),
            "host": socket.gethostname(),
            "runs_local": str(args.runs_local),
            "pseudo_dir": detect_pseudo_dir(args.runs_local),
            "exporter_python": sys.version.split()[0],
        },
        "filter": {
            "kind": "where" if args.filter else ("run-ids" if args.run_ids else "since"),
            "expression": args.filter or (str(args.run_ids) if args.run_ids else args.since),
        },
        "n_runs": len(runs_summary),
        "db_columns": cols,
        "runs": runs_summary,
    }
    (bundle_root / "manifest.json").write_text(
        json.dumps(manifest, indent=2, ensure_ascii=False), encoding="utf-8")

    print(f"[export] writing tarball {args.out} ...")
    args.out.parent.mkdir(parents=True, exist_ok=True)
    with tarfile.open(args.out, "w:gz") as tar:
        tar.add(bundle_root, arcname=bundle_root.name)

    size_mb = args.out.stat().st_size / 1024 / 1024
    print(f"[export] DONE")
    print(f"  bundle path : {args.out}")
    print(f"  bundle size : {size_mb:.1f} MB")
    print(f"  n_runs      : {len(runs_summary)}")
    print(f"  next step   : `gh release create <tag> --repo <owner>/<repo> {args.out}`")

    shutil.rmtree(staging, ignore_errors=True)


if __name__ == "__main__":

§6 import.py 全文

纯标准库 · Python 3.10+ · ~230 行 · 复制保存为 import.py

#!/usr/bin/env python3
"""bench run data importer — 把 export.py 出的 tar.gz 解到本地 runs_local/ + INSERT 到 state.db。

用法：
    python import.py \\
        --bundle /tmp/bench-bundle-weilin-baseline-2026-05-13.tar.gz \\
        --runs-local /home/shijie/bench/runs_local \\
        --pseudo-dir /home/shijie/bench/dft-agent-bench/pseudos/GBRV_USPP_PBE_v1.5 \\
        --on-conflict skip

冲突策略 (--on-conflict)：
  skip       run_id 已在 dest state.db → 跳过（默认）
  overwrite  覆盖：先 DELETE 旧行 + rm -rf 旧 run_dir → 再写新的
  fail       直接报错退出

路径重写：
  state.db 里的 `run_dir` 列被改成 `<--runs-local>/` 绝对路径
  workdir/pseudos 软链重新指向 `<--pseudo-dir>`
  pid 设 NULL，owner 保留源端值
"""
from __future__ import annotations
import argparse, json, os, shutil, sqlite3, sys, tarfile, tempfile
from pathlib import Path


def parse_args():
    p = argparse.ArgumentParser(description="Import a bench-bundle.tar.gz into local runs_local/ + state.db.")
    p.add_argument("--bundle", type=Path, required=True, help=".tar.gz produced by export.py")
    p.add_argument("--runs-local", type=Path, required=True,
                   help="destination runs_local/ dir (must contain state.db)")
    p.add_argument("--pseudo-dir", type=Path, required=True,
                   help="local pseudopotential dir (workdir/pseudos symlinks point here)")
    p.add_argument("--on-conflict", choices=("skip", "overwrite", "fail"), default="skip")
    p.add_argument("--dry-run", action="store_true",
                   help="show what would happen; don't write anything")
    return p.parse_args()


def extract_bundle(bundle: Path, dest: Path) -> Path:
    """Extract tar.gz to dest, return path to the bundle's top-level dir."""
    with tarfile.open(bundle, "r:gz") as tar:
        members = tar.getmembers()
        roots = {m.name.split("/", 1)[0] for m in members if "/" in m.name or not m.isdir()}
        tar.extractall(dest)
    if len(roots) != 1:
        sys.exit(f"[import] bundle should have exactly 1 top-level dir, got: {roots}")
    return dest / next(iter(roots))


def get_dest_columns(db_path: Path) -> list[str]:
    con = sqlite3.connect(str(db_path))
    cols = [r[1] for r in con.execute("PRAGMA table_info(runs)")]
    con.close()
    return cols


def existing_run_ids(db_path: Path, candidate_ids: list[str]) -> set[str]:
    if not candidate_ids:
        return set()
    con = sqlite3.connect(str(db_path))
    placeholders = ",".join("?" * len(candidate_ids))
    rows = con.execute(
        f"SELECT run_id FROM runs WHERE run_id IN ({placeholders})", candidate_ids
    ).fetchall()
    con.close()
    return {r[0] for r in rows}


def relink_pseudos(workdir: Path, pseudo_dir: Path):
    """Recreate workdir/pseudos symlink pointing to local pseudo_dir."""
    link = workdir / "pseudos"
    if link.exists() or link.is_symlink():
        link.unlink()
    link.symlink_to(pseudo_dir, target_is_directory=True)


def insert_row(con: sqlite3.Connection, dest_cols: list[str], row: dict, run_dir_new: str):
    """INSERT a single row, picking only columns that exist in dest schema."""
    use_cols, use_vals = [], []
    for c in dest_cols:
        if c == "run_dir":
            use_cols.append(c); use_vals.append(run_dir_new); continue
        if c == "pid":
            use_cols.append(c); use_vals.append(None); continue
        if c in row:
            use_cols.append(c); use_vals.append(row[c])
        # column in dest but not in source row → leave for DB default
    placeholders = ",".join("?" * len(use_cols))
    cols_sql = ",".join(use_cols)
    con.execute(f"INSERT INTO runs ({cols_sql}) VALUES ({placeholders})", use_vals)


def main():
    args = parse_args()
    db_path = args.runs_local / "state.db"
    for p, label in [(args.bundle, "bundle"), (db_path, "destination state.db"),
                     (args.pseudo_dir, "pseudo dir")]:
        if not p.exists():
            sys.exit(f"[import] {label} not found: {p}")

    print(f"[import] reading manifest from {args.bundle}")
    staging = Path(tempfile.mkdtemp(prefix="bench-import-"))
    try:
        bundle_root = extract_bundle(args.bundle, staging)
        manifest = json.loads((bundle_root / "manifest.json").read_text(encoding="utf-8"))
        if manifest["schema_version"] > 1:
            sys.exit(f"[import] bundle schema_version={manifest['schema_version']} too new; "
                     "upgrade import.py")
        print(f"[import] bundle: tier={manifest['tier']} schema={manifest['schema_version']} "
              f"source={manifest['source']['user']}@{manifest['source']['host']} "
              f"n_runs={manifest['n_runs']}")
        print(f"[import] destination runs_local : {args.runs_local}")
        print(f"[import] destination state.db   : {db_path}")
        print(f"[import] destination pseudo_dir : {args.pseudo_dir}")

        dest_cols = get_dest_columns(db_path)
        src_cols = set(manifest["db_columns"])
        common = [c for c in dest_cols if c in src_cols]
        src_only = sorted(src_cols - set(dest_cols))
        dest_only = sorted(set(dest_cols) - src_cols)
        print(f"[import] dest columns ({len(dest_cols)}): {', '.join(dest_cols[:6])}, ..., {dest_cols[-1]}")
        if src_only:
            print(f"[import] dropping {len(src_only)} source-only column(s): {src_only}")
        if dest_only:
            print(f"[import] dest-only column(s) (will get DB default or NULL): {dest_only}")

        # Read all rows from db_rows.jsonl
        rows = []
        for line in (bundle_root / "db_rows.jsonl").open(encoding="utf-8"):
            rows.append(json.loads(line))
        all_ids = [r["run_id"] for r in rows]
        existing = existing_run_ids(db_path, all_ids)

        n_imported = n_skipped = n_failed = n_overwritten = 0
        if args.dry_run:
            print("[import] *** DRY RUN — no changes will be made ***")
        else:
            con = sqlite3.connect(str(db_path))

        for i, row in enumerate(rows, 1):
            run_id = row["run_id"]
            dst_run_dir = args.runs_local / run_id

            if run_id in existing:
                if args.on_conflict == "skip":
                    n_skipped += 1
                    print(f"[import] {i:3d}/{len(rows)}  {run_id:60s} → SKIP (already in dest)")
                    continue
                if args.on_conflict == "fail":
                    sys.exit(f"[import] FAIL: {run_id} already exists in dest")
                # overwrite
                if not args.dry_run:
                    con.execute("DELETE FROM runs WHERE run_id=?", (run_id,))
                    if dst_run_dir.exists():
                        shutil.rmtree(dst_run_dir)
                n_overwritten += 1

            src_run_dir = bundle_root / "runs" / run_id
            if not src_run_dir.exists():
                print(f"[import] {i:3d}/{len(rows)}  {run_id:60s} → FAIL (missing in bundle)")
                n_failed += 1
                continue

            verb = "OVERWRITE" if run_id in existing else "imported"
            if args.dry_run:
                print(f"[import] {i:3d}/{len(rows)}  {run_id:60s} → would {verb.lower()}")
                continue

            dst_run_dir.parent.mkdir(parents=True, exist_ok=True)
            shutil.copytree(src_run_dir, dst_run_dir)
            workdir = dst_run_dir / "workdir"
            if workdir.is_dir():
                relink_pseudos(workdir, args.pseudo_dir)

            try:
                insert_row(con, dest_cols, row, str(dst_run_dir))
            except sqlite3.IntegrityError as e:
                print(f"[import] {i:3d}/{len(rows)}  {run_id:60s} → FAIL (sqlite: {e})")
                n_failed += 1
                shutil.rmtree(dst_run_dir, ignore_errors=True)
                continue

            n_imported += 1
            print(f"[import] {i:3d}/{len(rows)}  {run_id:60s} → {verb}")

        if not args.dry_run:
            con.commit()
            con.close()

        print(f"\n[import] DONE")
        print(f"  imported   : {n_imported}")
        print(f"  overwritten: {n_overwritten}")
        print(f"  skipped    : {n_skipped}")
        print(f"  failed     : {n_failed}")
        if args.dry_run:
            print("  (dry-run: nothing written)")

    finally:
        shutil.rmtree(staging, ignore_errors=True)


if __name__ == "__main__":
    main()

§7 反向使用（Pluto235 → weilin）

同一对脚本可以无修改地反向用。我（Pluto235）想把自己跑的 solverv3 27 题包给 weilin：

# 在 Pluto235 服务器：
python export.py \
    --runs-local /home/shijie/bench/runs_local \
    --filter "agent_id='solverv3' AND model='deepseek-v4-pro' AND created_at >= '2026-05-11T16:00'" \
    --tier lossy \
    --label "Pluto235 standard tier solverv3 27 runs" \
    --out /tmp/bench-bundle-pluto235-solverv3-27-2026-05-13.tar.gz

gh release create solverv3-standard27-2026-05-13 \
    --repo Pluto235/dft-agent-data-shares \
    --title "Pluto235 solverv3 (27 runs, standard tier)" \
    /tmp/bench-bundle-pluto235-solverv3-27-2026-05-13.tar.gz

# weilin 那边：
gh release download solverv3-standard27-2026-05-13 \
    --repo Pluto235/dft-agent-data-shares --dir /tmp/

python import.py \
    --bundle /tmp/bench-bundle-pluto235-solverv3-27-2026-05-13.tar.gz \
    --runs-local /home/weilin/bench/runs_local \
    --pseudo-dir /home/weilin/bench/dft-agent-bench/pseudos/GBRV_USPP_PBE_v1.5 \
    --on-conflict skip

因为 state.db 的列在两边 schema 上是同源（都从 wanweilin/dft-agent-bench-web-fresh fork）， import.py 会自动用 PRAGMA table_info 探测目标列，源端多出来的列（如 agent_id）只要 dest 也有就 INSERT； dest 多出来的列就给 NULL 或 DB 默认值。跨 fork 的微小 schema drift 不会让导入崩。

§9 故障排查

❌ "sqlite3.IntegrityError: UNIQUE constraint failed: runs.run_id"

原因：dest 已经有这个 run_id 但 --on-conflict 不是 skip。 解决：要么改 --on-conflict skip 跳过，要么 --on-conflict overwrite 强覆盖。

❌ "pseudo dir not found"

原因：传给 import.py 的 --pseudo-dir 路径在 dest 机器上不存在。 解决：先确认 ~/bench/dft-agent-bench/pseudos/ 下有 GBRV_USPP_PBE_v1.5 子目录； 没有就 cd ~/bench/dft-agent-bench && git pull（pseudos 跟着仓走）。

❌ web 界面 leaderboard 不显示新导入的 run

检查清单：

  1. sqlite3 ~/bench/runs_local/state.db "SELECT COUNT(*) FROM runs" 看 DB 行数是否增加
  2. ls ~/bench/runs_local/<run_id>/report.json 看文件确实拷过来了
  3. web 服务可能 cache 了 DB；bash ~/web/start.sh restart 重启一下
  4. 看 web backend log：tail -50 ~/web/logs/*.log | grep -i error

⚠ created_at 显示成奇怪的时间

原因：源端 created_at 存的是 ISO UTC 字符串（如 2026-05-08T14:08:52Z）， 但前端可能按本地时区渲染。如果你跟我时区不同（你 UTC+8，我 UTC+8——其实一样）， 看起来时间偏 0 小时；其他情况下偏 ±X 小时。不是 bug，是显示约定。

⚠ transcript.json 特别大（> 10 MB）

少数 SCF 卡死循环的 run，transcript 会膨胀到几十 MB。脚本不会主动截断， 但 web 前端可能渲染卡顿。导入完后这种 run 单独点进去会慢，是已知现象。 后续考虑给 import.py 加 --max-transcript-mb 10 选项做硬截断。

⚠ source-only / dest-only 列警告

正常：两边 fork 不同步，会有 schema drift。脚本会自动 drop 源端独有的列、 dest 独有的列写 NULL。如果某个列你觉得"必须保留"，确保 dest 端 ALTER TABLE 加上同名列再 import。

§10 未来自动化建议

用过 3-5 次之后，可以考虑这些升级：

  • 收进 web/scripts/，作为 bench-sync export/import 子命令，跟 web 一起版本管。
  • 增量同步：export 加 --since-bundle <prev.tar.gz>，只打这次 bundle 之后新增的 runs； import 加 --idempotent 自动跳过已存在的 ID。
  • 压缩升级：现在用 gzip，~500 MB；换 zstd 能压到 ~200-300 MB（需要 pip install zstandard）。
  • WebHook 集成：GH Release 发布时触发 webhook → 自动通知对方 import。
  • 校验和：manifest 里加 sha256 per-run，import 时校验防中途损坏。
  • 引用 wanweilin/dft-agent-bench 上游：bench 题目本身也跟着 git 走， 所以两边 task_id 永远一致；这块不用 sync。

§11 附录：full tier 切片 / HuggingFace dataset 方案

切片：当 full tier 超过 GH Release 单文件 2 GB

Lossy 一般够；只有想要完全 bit-exact 复现（重做 dos.x / 重画图）才需要 full。 Full 36 GB 不可能一个文件——可以按 task group 切：

# 切成 4 份：M1_T1 / M1_T2 / M2_T1 / M2_T2
for grp in M1_T1 M1_T2 M2_T1 M2_T2; do
    python export.py \
        --runs-local ~/bench/runs_local \
        --filter "task_id LIKE '${grp}_%' AND status='done'" \
        --tier full \
        --out /tmp/bench-bundle-full-${grp}-2026-05-13.tar.gz
done

# 每片上传成单独的 release asset
gh release create full-2026-05-13 --repo Pluto235/dft-agent-data-shares \
    /tmp/bench-bundle-full-M1_T1-2026-05-13.tar.gz \
    /tmp/bench-bundle-full-M1_T2-2026-05-13.tar.gz \
    /tmp/bench-bundle-full-M2_T1-2026-05-13.tar.gz \
    /tmp/bench-bundle-full-M2_T2-2026-05-13.tar.gz
